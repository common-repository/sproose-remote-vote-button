== Description ==

This plugin adds a remote vote button for the social search engine sproose to the bottom of each post.

This plugin is based on the obsocialbookmark plugin (http://www.oraclebrains.com/wordpress/plugin/ob_social_button)
created by Rajender Singh.

== Installation ==

1. Upload 'sproose-remote-vote-button.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to "Options"->"Sproose Remote Vote Button" and enable the button you want
